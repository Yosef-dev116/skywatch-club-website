Skywatch Club (CS2060 Website)
1) Theme: stargazing club pages (Home/Gallery/Plan/Contact).
2) Navigation: buttons are <a><button>...</button></a> on every page.
3) Links: internal links between pages + external links (NASA, Timeanddate) + mailto.
4) Images: 3 local SVGs in /images (nebula.svg, starpath.svg, mission.svg). Each has alt text.
5) Lists: Home + Plan pages use <ul> and <ol>.
6) Table: Plan page uses a <table> for meetup schedule.
7) Form: Contact page uses labels, text/email/tel/date/color/range, select, textarea, radio, checkbox, submit/reset.
8) CSS: external styles.css + internal <style> per page + one inline style on Home heading.
Files: index.html, gallery.html, plan.html, contact.html, styles.css, readme.txt, images/.
